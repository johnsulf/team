# Team Bergen Diskgolf

Nettside for Team Bergen Diskgolf – en gjeng diskgolf-entusiaster fra Bergen og omegn.

## Teknologi

- [Astro](https://astro.build) – statisk nettstedsgenerator
- [Tailwind CSS](https://tailwindcss.com) – utility-first CSS
- [daisyUI](https://daisyui.com) – komponentbibliotek for Tailwind

## Utvikling

```bash
# Installer avhengigheter
npm install

# Start utviklingsserver
npm run dev

# Bygg for produksjon
npm run build

# Forhåndsvis produksjonsbygg
npm run preview
```

## Hosting

Siden hostes på [Cloudflare Pages](https://pages.cloudflare.com) og er tilgjengelig på [teambergen.no](https://teambergen.no).
